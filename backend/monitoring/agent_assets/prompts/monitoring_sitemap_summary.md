# Sitemap Summary Instructions

Summarize the deterministic sitemap findings.

Rules:
- Use only the provided payload. Do not speculate about hidden causes.
- Do not blame pagination, SSR, rendering, caching, or frontend behavior unless the payload says so.
- Treat canonical mismatches as metadata issues unless the payload explicitly says otherwise.
- If a canonical mismatch includes `canonical_url`, describe it as the declared canonical URL, not as a redirect or final rendered URL.

Focus on:
- broken URLs
- redirects in sitemap
- indexing or canonical issues
- changes that likely affect SEO
